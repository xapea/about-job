#1
躲在角落的宅宅
資料工程師
躲在角落默默把網站介面刻出來
擅長使用鑄鐵鍋煎牛排
喜歡買IKEA的傢俱來重新改裝
關於我在做的事
上週有個朋友打電話給我，說她認真考慮跟我一樣轉行當軟體工程師，想要多理解我的工作內容。首先說明一下，coding 相關的職業非常多：你可以寫手機的 app、寫網頁、開發人工智能、做金融業的預測模型、做機器人（包含工廠用來大量生產各種產品的機器）、開發自動駕駛技術等等等。太多了。程式語言存在的目的就是做自動化、減少人工作業時間，所以運用的範圍無遠弗屆。網頁開發工程師，顧名思義就是寫網頁的人。（195個字）
想對大家說的話
在我心目中寫程式一直是一件很困難的事情。遙想大學時代學C++，我就因為覺得太難理解 code 而放棄學習一個我覺得很有趣的科目。 現在有點不同了。Youtube 的發達、網路社群的發達，使得學習任何東西都沒有那麼難。再怎麼愚蠢或困難的問題，google一下就會有答案。所以是的，我相信任何人都能夠自學寫程式，問題在於你可以學到多好的程度？可以好到拿來當吃飯的工具嘛？也許你只會幾句日文，足以讓你在日式餐廳打工；但如果你想當日文老師，那你就需要會複雜的文法甚至日本文學。（234個字）
